Note on Framesets

The <frameset> and <frame> tags were deprecated in HTML5. Modern web development uses <iframe> instead for embedding content.

Legacy Frameset Example (For Reference Only):

<frameset cols="25%,75%">
    <frame src="menu.html" name="menu">
    <frame src="content.html" name="content">
</frameset>
